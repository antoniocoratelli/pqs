

conversion_factor

represents the scaling of the concrete-quantity value from the concrete-base-quantity value

distinguish conversion factors which have same exponent and multiplier using custom_conversion_factor

conversion factor sub concepts

   basic_conversion_factor      -> The exponent is 0 and multiplier is 1
   integer_conversion_factor    -> the exp is an integer and multiplier is 1
   rational_conversion_factor   -> the exp is a rational and multiplier is 1
   custom_conversion_factor     -> has a name, rational exp and rational multiplier
                                                     
Informal rule : favour no custom conversion factor in terms of length of error message

operations
   
ratio get_exponent(conversion_factor) ;  return the base 10 exponent of the scaling factor
ratio get_multiplier(conversion_factor) ; return the multiplier of the conversion factor
bool is_custom(conversion_factor) ; true if it is a custom conversion factor
unit_system get_unit_system(conversion_factor) : get the unit system this conversion factor belongs to
