Compatibility

The classes have been tested on Microsoft Visual C++.Net 7.1 and gcc 3.2. They are designed to follow the current C++ standard, so should work on any (closely) conforming compiler.
(they will not compile on VC++6 or other older compilers which do not implement template members, and are not recommended for use with compilers that do not implement long integer compile time arithmetic.)

Dependencies

The default implementation requires the boost libraries. I have provide an implementation of the required headers if you dont have the boost libs, but use of the boost libs is recommended.

Installation 

The files are packed with their directories, so you merely need to unzip them in your chosen location. The class is entirely contained in the headers so there is no need to create or use object libraries. Once installed use is as simple as including the required headers. See the File list for more information, particularly if you do not have the boost libraries.
