


**base_dimension**
some type that is ordered with respect to all other base_dimension used in the same dimension
The type must implement equality

base_dimension operations  T is some type

is_base_dimension<T>

bdL is some type for which is_base_dimension<bdL> is true
bdR is some type for which is_base_dimension<bdR> is true

less_than_comparable<bdL,bdR> -> bool
equality_comparable<bdL,bdR> -> bool

less_than<bdL,bdR> -> bool
equal_to<bdL,bdR> -> bool

are_ordered<bdL,bdR> = less_than_comparable<bdL,bdR> && equality_comparable<bdL,bdR>
